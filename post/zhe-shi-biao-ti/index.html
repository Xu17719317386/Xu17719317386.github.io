1.svn与git的区别
	SVN：一台服务器，多台主机，一个局域网内很好用
	Git:分布式版本控制系统，没有服务器
	
2.创建本地仓库
	git GUI Here:git init
	小乌龟：直接创建本地仓库

3.本地仓库  add   暂存区   commit提交    本地仓库
    本地仓库  push推送   远程仓库remote
    远程仓库remote  clone克隆   本地仓库
    
4.含.git文件夹为工作目录，要提交的文件都放在这一同级目录
	java项目：
		.idea  配置文件，不用提交
		lib：jar包，提交
		out：动态生成的java字节码文件.class文件，不用提交
		serc:src下的java代码就在这里，必须提交
		
		不用提交的文件选择递归忽略即可
	
5.小乌龟使用
	版本库浏览器：本地仓库
	显示日志：选中2个比较版本差异
	删除文件：
		直接本地删除，版本库浏览器中还在，得提交后才有效
		删除并保留本地副本：只删除版本库浏览器
